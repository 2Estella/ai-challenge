# 🔍 Code Review Checklist Prompt

## 🎯 목적
코드 리뷰 및 품질 점검을 자동화/체크리스트 기반으로 수행

## 체크 항목
- [ ] 불필요한 상태 관리 제거?
- [ ] zustand store → selector 최소화/shallow 사용?
- [ ] 재사용 가능한 컴포넌트 분리?
- [ ] Props 정의(필수/옵션/기본값) 명확?
- [ ] 네이밍 일관성 유지?
- [ ] SCSS 규칙(BEM, 모듈) 준수?
- [ ] 타입/에러 처리 누락 없음?
- [ ] 접근성(a11y) 반영? (role/label/focus trap)
- [ ] 테스트 케이스 포함 여부?

## 출력 형식
- 리뷰 코멘트 리스트 (✅ 통과 / ⚠️ 개선 / ❌ 문제)
- 각 항목별 짧은 설명 포함
